# brisma-plugins

Claude Code plugin marketplace by brisma.

## Installation

```bash
/plugin marketplace add brisma/claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `psx-spx` | Complete PSX hardware reference (CPU, GPU, SPU, DMA, CDROM, BIOS, and more) |

### Install a plugin

```bash
/plugin install psx-spx@brisma-plugins
```
